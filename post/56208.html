记录学习scala过程中遇到的错误

already defined as object

出现错误***is already defined as object,网上很多的答案是修改设置是file->project structure->modules->source里可以修改文件是否为source定义src目录下Java和scala作为源码包,如果该操作无效, 可以修改pom文件中<sourceDirectory>src/main/scala</sourceDirectory>指定源码位置.

ide依赖包问题

工程中各依赖检查没有出错,但是在object中会抛出无法寻找依赖的问题.可以更新idea到最新版解决(难得遇到idea的bug).

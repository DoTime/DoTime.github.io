资源使用

  1. 不死鸟的分享: 免费图床对比.
  2. sm图床: 免费的,nexmoe主题使用的图床,速度很慢.
  3. 腾讯图床: 免费额度限制,需实名认证.
  4. 自建图床: 使用github资源目录解决图床问题:使用raw域名/用户名/仓库名/分支/文件路径来解决,例如本页面使用https://raw.githubusercontent.com/DoTime/dotime.github.io/master/asset/img/goods/watch1.jpg该资源为题图.

迁移指南

  1. 新建文件夹,初始化hexo, npx hexo init
  2. clone主题文件
    • cd themes
    • git clone https://github.com/nexmoe/hexo-theme-nexmoe.git nexmoe
  3. clone代码git clone https://github.com/DoTime/MyblogCode.git
  4. 安装依赖
    • npm install
    • npm install hexo-deployer-git --save
  5. 主题必须安装计数插件npm i --save hexo-wordcount
  6. 安装git发布插件npm install hexo-deployer-git --save
  7. 文章链接唯一化插件npm install hexo-abbrlink --save
  8. 代码压缩插件npm install hexo-all-minifier --save
  9. 博客源码重新关联远程git仓库.

零散记录

  1. layout参数标识当前md文件渲染的方式. 在当前nexmoe主题下, py不会显示题图, friend会显示题图,默认为post.
  2. 文章起始部分使用<!--more -->标签进行概要的截断.
  3. 使用阿里的图标库:登陆阿里矢量库注册=>图标管理=>新建项目=>编辑项目页面 Font Family参数修改为nexmoefont=>添加新图标到项目=>生成的css地址添加到主题配置文件iconlib参数的value部分=>在使用图标部分引用在矢量库的class.
  4. 删除题图字数统计和阅读时间标签的方法: 全局搜索分钟,删除theme文件夹下生成字和分钟的标签代码.
  5. 缩小copyright字体方法:定位themes\nexmoe\source\css\style.styl文件,在nexmoe-copyright样式加入font-size:0.56em;字体大小配置.
  6. idea识别编辑ejs文件的方法: setting=>File Types => 查找HTML=>在js的Registered栏中添加*.ejs=>确认.
